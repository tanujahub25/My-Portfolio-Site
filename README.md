# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. It showcases my skills, projects, and contact information in a clean, responsive layout. The goal of this site is to present myself professionally as a web developer.

## 🚀 Features

- Responsive design (works on all screen sizes)
- Smooth navigation with scroll behavior
- Sections: Home, About, Projects, Skills, Contact
- Interactive buttons and icons
- Linked social media profiles

## 💡 Tech Stack

- HTML5
- CSS3
- JavaScript
